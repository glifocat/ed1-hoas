# ESPHome Devices Database Submission

This folder contains files prepared for contributing the Citilab ED1 to the [ESPHome Devices Database](https://devices.esphome.io/).

## Folder Structure

```
citilab-ed1/
├── index.md              # Device page with front matter
├── citilab-ed1.yaml      # Minimal standalone config
├── ed1-board.png         # Main board image
├── ed1-front-angle.png   # Front angle view
├── ed1-back-angle.png    # Back angle view
└── LICENSE               # Image license (CC BY-SA 4.0)
```

## How to Submit

1. Fork the [esphome/esphome-devices](https://github.com/esphome/esphome-devices) repository
2. Copy the `citilab-ed1/` folder to `src/docs/devices/`
3. Create a pull request
